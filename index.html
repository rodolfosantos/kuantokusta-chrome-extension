kuantokusta-chrome-extension

KuantoKusta price alert Chrome Extension.

Description

This Chrome Extension allows you to follow the lowest prices of your favorite products from KuantoKusta website.

How it works

  • Install Extension using kuantokusta-chrome-extension.crx file
  • Go to KuantoKusta and search for the product you would like to follow the price
  • Go to extension menu and click on Subscribe current item!
  • Now this product and its lower available price appear in products list
  • When the price drops, will be notified with a popup
